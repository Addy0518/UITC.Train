5. 使用 NVM 安裝 Node.js
6. GIT
7. Postman

## 綜合練習

請開設一個前後分離專案，
並設計一個簡易 2-3 張表，
後端 API 請撰寫 Dapper + 三層式，
前端請使用 Vue.js + Vite 建立網站，
套件請使用 PrimeVue 作為元件庫，
並請上傳至 GITHUB 上

## Git commit message 規則

| 格式      | 說明     |
| --------- | -------- |
| feat:     | 新增功能 |
| fix:      | 修復 bug |
| refactor: | 重構     |
| chore:    | 雜項     |
| test:     | 測試     |
| docs:     | 文件     |
| style:    | 格式     |
| update:   | 更新     |
| delete:   | 刪除     |
